Будівельні теги(header,main,footer та div)

Header

Header- це будівельнй тег який зазвичай викристовуєтся на нульовому рівні вкладиності і відповідає за шапку сайту.

Нажаль синтаксис не покажу так як редактор сприйме його як частину коду(

Main

Main- це будівельнй тег який зазвичай викристовуєтся на нульовому рівні вкладиності і відповідає головну частину сайту.

Footer

Footer- це будівельнй тег який зазвичай викристовуєтся на нульовому рівні вкладиності і відповідає за підвал сайту.

Теги для контенту(h1-h6,p,span/mark,a,img,br)

H1-H6

H1-H6- це група тегів для заголовків

P

P- це тег для абзацу

Span/mark

Span/mark- це теги для виділеня частини тексту

A

A- це тег для створення посилання

IMG

IMG- це тег для додавання картинок

BR

BR- це тег для переносу рядка на новий рядок
